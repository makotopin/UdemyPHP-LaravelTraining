# PHPからLaravelまで サーバーサイドをとことんやってみよう
## 93. エラーメッセージの日本語化対応

先日インストールしたLaravel Breezeによる認証等の機能を確認
例えば、新規登録（Register）でパスワードを１文字で登録しようとするとエラーメッセージが表示される。
ただ、このエラーメッセージが英語表記で出てきてしまうので日本語に直す方法を確認する。

1. マニュアルの言語ファイルをlang/ja/配下にそれぞれ配置する
2. validation.phpの'attributes'に追記する
3. ja.jsonファイルを作成する。
※本当はconfigも必要なのだが、それは以前変更しているのでOK

lang/ja/validation.phpのカステムバリデーションで、:attributeの部分カスタムする
```'attributes' => [
  'password' => 'パスワード'
] ,```と入力するすることで日本語になる。

また、ja.jsonファイルには
```{
    "Whoops! Something went wrong.": "何か問題が発生しました。"
}```
と入力することで、初期表示メッセージを変更することができる。
